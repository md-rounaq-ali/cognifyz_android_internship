## 📋 Objective
Create an app with a simple form and form submission.

## ✅ Requirements Implemented
- Form layout with input fields: **Full Name**, **Email Address**, **Phone Number**, **Role** (dropdown), **Message**
- Submit button that validates all fields before submitting
- Displays a **SnackBar** (Toast equivalent) on invalid input
- Animated success screen with checkmark animation after submission
- **Validation rules:**
  - Name: minimum 2 characters
  - Email: regex pattern match
  - Phone: minimum 10 digits
  - Message: minimum 10 characters
  - Terms & Conditions checkbox must be checked
- 2-second loading animation during form submission

## 🚀 How to Run
```bash
flutter run                  # Android device/emulator
```
