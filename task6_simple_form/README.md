# Form Validation System

A robust Flutter application showcasing complex form handling, input validation, and user feedback mechanisms. Developed as part of the Android Development portfolio.

## ✨ Features

- **Comprehensive Validation:** Regex-based email verification, character limits, and exact digit matching for phone numbers.
- **Input Types:** Supports text fields, numerical keyboards, dropdowns, and checkboxes.
- **Feedback UI:** Displays localized error messages and contextual SnackBars upon invalid attempts.
- **Async Submission:** Simulates network latency with a loading state before triggering a success animation.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Design System:** Material 3

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0.0 or higher)

### Installation & Run

1. Navigate to the project directory:
   ```bash
   cd task6_simple_form
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
