## 📋 Objective
Modify the app to fetch and display data dynamically from the internet.

## ✅ Requirements Implemented
- Fetches **live data** from `https://jsonplaceholder.typicode.com/users`
- Displays data dynamically in a **ListView** (equivalent of RecyclerView)
- Loading spinner while data is being fetched
- Error state with retry button if network fails
- Real-time **search filter** across fetched users
- Refresh button to re-fetch data on demand

## 🚀 How to Run
```bash
flutter run                  # Android device/emulator
```
> ⚠️ Requires an active internet connection to fetch data.

## 📦 Dependencies
- `google_fonts` – Outfit font
- `http` – Network requests
