# REST API Data Fetcher

A network-aware Flutter application that performs asynchronous HTTP requests, parses JSON data, and handles varying network states. Developed as part of the Android Development portfolio.

## ✨ Features

- **Asynchronous Networking:** Uses the `http` package to securely fetch live JSON data.
- **State Handling:** Implements comprehensive Loading, Success, and Error UI states.
- **Pull-to-Refresh:** Allows users to manually trigger data re-fetching.
- **Live Search:** Client-side filtering mechanism to instantly search fetched payloads.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Networking:** `http` package

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0.0 or higher)
- Active internet connection

### Installation & Run

1. Navigate to the project directory:
   ```bash
   cd task5_fetch_data
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
