# Dynamic List Display

A performant Flutter application demonstrating advanced list rendering, filtering, and expandable UI components. Developed as part of the Android Development portfolio.

## ✨ Features

- **Efficient Rendering:** Utilizes `ListView.builder` for highly performant scrolling of large datasets.
- **Real-time Filtering:** Integrated search bar that dynamically filters the list state.
- **Expandable Cards:** Custom card widgets with `SizeTransition` for smooth expand/collapse animations.
- **Categorization:** Visual badges and chips to represent item roles and tags.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Design System:** Material 3

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0.0 or higher)

### Installation & Run

1. Navigate to the project directory:
   ```bash
   cd task3_list_display
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
