# Hello World App

A highly polished Flutter application demonstrating fundamental UI rendering, cross-platform responsiveness, and custom entrance animations. Developed as part of the Android Development portfolio.

## ✨ Features

- **Animated Entrance:** Smooth fade, scale, and slide transitions triggered on app initialization.
- **Custom Typography:** Integrated `google_fonts` (Outfit) for premium text styling.
- **Responsive Layout:** Dynamically adapts to various screen sizes without overflow (Mobile & Web).
- **Modern UI/UX:** Utilizes gradients, drop shadows, and Material Design 3 principles.

## 🛠️ Tech Stack

- **Framework:** Flutter
- **Language:** Dart
- **Design:** Material 3

## 🚀 Getting Started

### Prerequisites
- Flutter SDK (v3.0.0 or higher)
- Android Studio / VS Code

### Installation & Run

1. Navigate to the project directory:
   ```bash
   cd task1_hello_world
   ```
2. Install dependencies:
   ```bash
   flutter pub get
   ```
3. Run the application:
   ```bash
   flutter run
   ```
